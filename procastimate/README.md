# ProcastiMate

ProcastiMate is a Chrome extension combining a Pomodoro timer with a site blocker to help you stay focused.

## Features
- Pomodoro timer with start/pause/reset
- Custom durations
- Site blocker that blocks listed sites when a work session is running
- Options page to manage blocked sites

## Install (developer mode)
1. Clone or download this repo
2. Open Chrome → chrome://extensions/
3. Enable Developer mode
4. Click Load unpacked and select the project folder

## Usage
- Open the extension popup to start/pause the pomodoro timer
- Add domains to the options page (one per line). During an active work session those domains will show a full-page overlay.
